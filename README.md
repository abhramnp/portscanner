# portscanner

Description
------------------------------------------

A Python-based port scanner with a graphical interface that performs TCP port scanning, basic service detection, and banner grabbing for version identification. Inspired by tools like Nmap, this project demonstrates core networking and>
Features
•       Scan custom port ranges
•       Multithreaded scanning
•       Service detection using system mapping
•       Banner grabbing for version detection
•       Progress bar with real-time updates
•       GUI interface using Tkinter
•       CLI execution (portscanner command)


Installation
------------------------------------------
1. Clone the repository
git clone https://github.com/your-username/port-scanner.git
cd port-scanner

2. Install requirements
pip install -r requirements.txt

3. Run GUI
python3 portscan.py

Run as Command (Linux)

chmod +x portscan.py
sudo cp portscan.py /usr/local/bin/portscanner
Run: portscanner

Comparison with Nmap
-This tool provides basic scanning and banner grabbing, while Nmap offers advanced service detection, OS detection, and multiple scan types.
